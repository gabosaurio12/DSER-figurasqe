
== Plan de Trabajo Inicial

El primer acercamiento al proyecto, será un análisis sobre las necesidades tecnológicas requeridas para su buen funcionamiento e integración. Para continuar con una fase de diseño para disminuir los percances técnicos que pueden llegar a haber.

=== Actividades Principales

* Documento de análisis de requisitos
* Documento de diseño
** Diagrama de clases
** Diagrama de componentes
** Diagrama de despliegue
** Diagrama de secuencia de las funcionalidades más importantes
* API de sesiones
* API de métrticas
* API de Detección de Manos
* Cliente web
